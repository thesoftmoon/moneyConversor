## Currency Converter

This is a currency converter in console way, where you select a combo, currency to currency, and an amount to convert, then you receive the final conversion and the app reset to the menu, in the menu you can select the option 9 and get a log of the current movements or conversion with a time stamp or select the exit option.

This app is developed in Java 17, consuming the [exchangerate API](https://www.exchangerate-api.com/) of all the currencies showed in the menu
